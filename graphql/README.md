# Graphql

Endpoint is available /graphql

Schema File is at `graphql/schema.graphql`

Config file at `config/graphql.php`

## Demo

```json
{
    contact(id: 1) {
      title
      email
      phone_number
      status
    }
    contacts {
      status
    }
}
```
